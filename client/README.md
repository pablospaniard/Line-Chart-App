# Audio Decoder App

## You can check app here: https://salty-forest-88930.herokuapp.com/

(it could take up to 30 seconds to start app)

### Start the project:

- clone this repo
- `yarn install`
- `yarn start`

### Tests

- `yarn test`

## Important Note:

you will need to have a CORS extension enabled on your browser to donwload mp3
file.
